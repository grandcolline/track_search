Track Search

Rustで作成された、楽曲を検索して情報を提供するWEBアプリケーション

起動方法や運用などについては READMEに、 アーキテクチャや実装方法などについては RustDoc にドキュメントとしてまとめる。

🤖 アーキテクチャ

5つの層(Layer)から構成される。

NameNote
エンティティ層ビジネスロジックを定義する層
ユースケース層ビジネスフロー/アプリケーションロジックを定義する層。ポートも持つ。
アプリケーション層ユーザとユースケースをつなぐ層。ユースケースの事項を行う。
アダプター層ユースケース層で定義されたポートの実装を持つ層
ドライバー層アプリケーションとアダプターの構築を行う層

依存関係

           ENTITY
              ↑
           USECASE
              ↑
   APPLICATION / ADAPTER
              ↑
            DRIVER
  • アプリケーション層とアダプター層はそれぞれドライバー層から参照され、ユースケース層に依存するが、互いには依存し合わない。
  • 各レイヤーはsrc直下にモジュールとして配置する。
  • レイヤー内の依存関係は、親モジュールが子モジュールに依存するような関係となる。
